# Massnahmen erstellen und verwalten

Massnahmen sind **konkrete Aktionen**, die du als Reaktion auf Einschätzungen definierst. Sie helfen dir, Risiken zu minimieren, Schwächen zu beheben, Chancen zu nutzen oder Stärken auszubauen.

## Übersicht

### Was ist eine Massnahme?

Eine Massnahme beschreibt:
- **Was** getan werden muss
- **Wer** dafür verantwortlich ist
- **Bis wann** es erledigt sein soll
- **Wie der Stand** der Umsetzung ist
- **Wer kontrolliert**, ob die Massnahme wirksam ist

### Massnahmen-Status

| Status | Bedeutung |
|--------|-----------|
| **geplant** | Massnahme ist definiert, aber noch nicht gestartet |
| **laufend** | Massnahme wird derzeit umgesetzt |
| **in Arbeit** | Massnahme ist aktiv in Bearbeitung |
| **abgeschlossen** | Massnahme wurde erfolgreich umgesetzt |
| **verschoben** | Umsetzung wurde auf einen späteren Zeitpunkt verschoben |

## Massnahmen anzeigen

### Alle Massnahmen

**Navigation:** IKS → Alle Massnahmen

Die Massenansicht zeigt alle Massnahmen, auf die du Zugriff hast. Standardmässig werden nur **aktive Massnahmen** (geplant, laufend, in Arbeit) angezeigt.

**Filtermöglichkeiten:**
- Nach **Status** filtern (geplant, laufend, in Arbeit, abgeschlossen, verschoben)
- Nach **überfälligen** Massnahmen filtern (Frist in der Vergangenheit)
- Nach **Verantwortung** filtern
- Nach **Organisationseinheit** filtern

### Meine Massnahmen

**Navigation:** IKS → Meine Massnahmen

Diese Ansicht zeigt nur die Massnahmen, die **dich direkt betreffen**:
- Massnahmen, bei denen du als **verantwortliche Person** eingetragen bist
- Massnahmen, bei denen du einer **verantwortlichen Funktion** angehörst
- Massnahmen, bei denen du als **kontrollierende Person** eingetragen bist
- Massnahmen, bei denen du einer **kontrollierenden Funktion** angehörst

> **💡 Tipp:** Die Ansicht "Meine Massnahmen" ist ideal für den täglichen Überblick. Du siehst sofort, welche Massnahmen deine Aufmerksamkeit erfordern.

## Massnahme erstellen

### Aus einer Einschätzung heraus

Der häufigste Weg, eine Massnahme zu erstellen:

1. Öffne die Einschätzung, zu der die Massnahme gehört
2. Scrolle zum Abschnitt **"Massnahmen"**
3. Klicke auf **"+ Massnahme"**
4. Fülle das Formular aus
5. Klicke auf **"Speichern"**

Die Verknüpfung zur Einschätzung wird automatisch gesetzt.

### Direkt über die Massnahmenliste

1. Navigiere zu **IKS → Alle Massnahmen**
2. Klicke auf **"+ Massnahme"**
3. Fülle das Formular aus
4. Klicke auf **"Speichern"**

### Aus einem anderen Objekt heraus

Massnahmen können auch aus Prozessen, IMS-Objekten und anderen verknüpften Elementen erstellt werden.

### Formularfelder

| Feld | Pflicht | Beschreibung |
|------|---------|--------------|
| **Titel** | Ja | Kurze, aussagekräftige Bezeichnung der Massnahme |
| **Beschreibung** | Nein | Detaillierte Beschreibung der Massnahme |
| **Frist** | Nein | Termin, bis wann die Massnahme umgesetzt sein soll |
| **Periodizität** | Nein | Regelmässige Durchführung (z.B. 1× jährlich) |
| **Status** | Ja | Aktueller Umsetzungsstand |
| **Verantwortliche Funktion** | Nein | Funktion, die für die Umsetzung verantwortlich ist |
| **Verantwortung** | Nein | Person, die für die Umsetzung verantwortlich ist |
| **Kontrollierende Funktion** | Nein | Funktion, die die Wirksamkeit kontrolliert |
| **Kontrolle durch** | Nein | Person, die die Wirksamkeit kontrolliert |
| **Beteiligte Organisationseinheiten** | Nein | Welche Abteilungen betroffen sind |

> **💡 Tipp:** Definiere sowohl eine **verantwortliche** als auch eine **kontrollierende** Person oder Funktion. So stellst du das Vier-Augen-Prinzip sicher.

## Detailansicht einer Massnahme

Wenn du eine Massnahme öffnest, siehst du:

### Kopfbereich
- **Titel** der Massnahme
- **Navigation**: Link zu "Meine Massnahmen"
- **Aktionen**: Bearbeiten, Löschen, Zurück
- **Dropdown-Menü**: Checkliste hinzufügen, Verknüpfungen hinzufügen

### Detailtabelle
- **Gehört zu**: Verknüpftes Objekt (z.B. Einschätzung)
- **Status**: Aktueller Umsetzungsstand
- **Verantwortung**: Verantwortliche Person mit Avatar
- **Beteiligte Organisationseinheiten**: Betroffene Abteilungen
- **Verantwortliche Funktion**: Verantwortliche Funktion mit Gruppenmitgliedern
- **Kontrollierende Person**: Wer kontrolliert
- **Kontrollierende Funktion**: Welche Funktion kontrolliert
- **Periodizität**: Regelmässigkeit der Durchführung
- **Frist**: Umsetzungstermin
- **Beschreibung**: Detaillierte Massnahmenbeschreibung
- **QR-Code**: Für schnellen Zugriff per Smartphone

### Checklisten
Falls vorhanden, werden verknüpfte Checklisten angezeigt. Diese helfen dir, die Umsetzung Schritt für Schritt abzuarbeiten.

### Kontrollen
Eine Tabelle aller durchgeführten Kontrollen mit:
- Datum
- Kontrollperson
- Kommentar
- Status (erledigt, in Arbeit, abgelehnt, erledigt mit Mangel)
- Beilagen (Checklisten, Dateien)

### Kommentare
Diskussionsbereich für Austausch zwischen Verantwortlichen und Kontrollierenden.

1. Öffne die Massnahme
2. Klicke auf das **Stift-Symbol** oben rechts
3. Bearbeite die gewünschten Felder
4. Klicke auf **"Speichern"**

## Periodische Massnahmen

Massnahmen können mit einer **Periodizität** versehen werden. Das bedeutet, dass die Massnahme regelmässig durchgeführt werden muss.

### Periodizität einstellen

1. Bearbeite die Massnahme
2. Wähle im Feld **"Periodizität der Durchführung"** die gewünschte Frequenz
3. Setze eine **Frist** für die erste Durchführung
4. Speichere

### Wie funktioniert es?

- Die Periodizität definiert, wie oft pro Jahr die Massnahme durchgeführt werden muss
- Nach Abschluss wird automatisch die nächste Frist berechnet
- Beispiel: "2× pro Jahr" → alle 6 Monate eine neue Frist

### Beispiele

| Periodizität | Intervall |
|-------------|-----------|
| 1× pro Jahr | Alle 365 Tage |
| 2× pro Jahr | Alle ~183 Tage |
| 4× pro Jahr | Alle ~91 Tage (quartalsweise) |

## Verknüpfungen und Checklisten

### Verknüpfungen hinzufügen

1. Öffne die Massnahme
2. Klicke auf das **Drei-Punkte-Menü** oben rechts
3. Wähle **"Verknüpfungen hinzufügen"**
4. Suche und wähle die zu verknüpfenden Objekte

### Checkliste hinzufügen

1. Öffne die Massnahme
2. Klicke auf das **Drei-Punkte-Menü** oben rechts
3. Wähle **"Checkliste hinzufügen"**
4. Erstelle Checklistenpunkte für die Umsetzung

> **💡 Tipp:** Checklisten werden bei der Erstellung einer Kontrolle automatisch kopiert. So hat jede Kontrolle ihre eigene Checkliste zum Abhaken.

---

## Überfällige Massnahmen

Eine Massnahme gilt als **überfällig**, wenn:
- Eine Frist gesetzt ist
- Die Frist in der Vergangenheit liegt
- Der Status noch nicht "abgeschlossen" ist

### Überfällige Massnahmen finden

- **IKS → Alle Massnahmen**: Filtere nach "überfällig"
- **IKS → Meine Massnahmen**: Klicke auf "Überfällig" Filter
- **Dashboard**: Überfällige Massnahmen werden im Dashboard-Widget angezeigt

## Häufige Fragen

### Kann eine Massnahme mehreren Einschätzungen zugeordnet sein?

Ja, über **Verknüpfungen** kannst du eine Massnahme mit mehreren Einschätzungen verbinden. Die direkte Zuordnung ("Gehört zu") verweist auf ein einzelnes Objekt, aber zusätzliche Verknüpfungen sind möglich.

### Wer sieht welche Massnahmen?

Das hängt von den Berechtigungen ab:
- **IKS (Lesen)**: Alle Massnahmen sichtbar
- **Massnahmen (Lesen)**: Alle Massnahmen sichtbar
- **Verantwortliche/Controller**: Nur zugewiesene Massnahmen
- Mehr dazu: [Berechtigungen]({{< ref "berechtigungen" >}})

### Was passiert, wenn ich eine Massnahme lösche?

Alle zugehörigen **Kontrollen** und **Kontrolldateien** werden ebenfalls gelöscht. Lösche Massnahmen daher nur, wenn du sicher bist.

### Kann ich den Verantwortlichen nachträglich ändern?

Ja, bearbeite die Massnahme und ändere das Feld "Verantwortung" oder "Verantwortliche Funktion".
